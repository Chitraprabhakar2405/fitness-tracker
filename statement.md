## Problem Statement

A lot of people struggle to consistently track their workouts, resulting in lack of awareness of their fitness progress and insufficient motivation to maintain healthy routines. Existing solutions can be complex or require advanced technical skills, making it challenging for beginners and everyday users to record daily exercise and analyze basic fitness trends.

## Scope of the Project

This project provides a  Fitness Tracker application that allows users to store, view, and summarize their workouts. The solution covers:
- Recording each workout with activity name aas desired, duration of each workout (in minutes), and calories burned after every workout.
- Managing all workout data in one session
- Providing summary of workout (total duration and calories)
- Simple, interactive command-line interface.

The project is designed for core functionality and code clarity, this can be extended to include data determination, more detailed analytics, input validation, and user profile management in future versions.

## Target Users

- Students and individuals who are beginning their fitness journeys.
- Fitness enthusiasts who like lightweight, distraction-free tools for fitness tracking.
- Anyone looking to track exercise from the command line with minimal setup.

## High-Level Features

- it adds a workout (input name, duration, and calories burned).
- Shows all recorded workouts in a list format.
- It can calculate and show the total workout duration and calories burned.
- It has simple menu-driven navigation
- It handles invalid inputs and empty log states gracefully
